Natural CLI - Examples

If you are looking for the main page, please go here. Also can be interesting to undersand the code read the Javadoc.

Example 1. The first two commands: 


       // Create the command
       Command showDateCommand =
       new Command(
         "show date", 
         "Shows the current date and time", 
         new ICommandExecutor ()
         {
           public void execute(Object[] params ) 
           {  System.out.println(new java.util.Date().toString());  }
         }		
       );
       Command helloWorldCommand =
       new Command(
         "hello world", 
         "Says hello.", 
         new ICommandExecutor ()
         {
           public void execute(Object[] params ) 
           {  System.out.println("Hello world!");  }
         }		
       );
       // Create the set of commands
       Set cs = new HashSet();
       cs.add(showDateCommand);
       cs.add(helloWorldCommand);
       // Execute
       new NaturalCLI(cs).execute(args);

Example 2. A command with a parameter: 


       Command showDateCommand =
         new Command(
           "hello world <string>", 
           "Says hello to the world and especially to some one.", 
           new ICommandExecutor ()
           {
             public void execute(Object[] params ) 
             {  System.out.println("Hello world! And hello especially to "+params[0]);  }
           }		
       );

Example 3. An optional parameter: 


       Command showDateCommand =
         new Command(
           "hello world [<name:string>]", 
           "Says hello to the world and, why not, especially to some one.", 
           new ICommandExecutor ()
           {
             public void execute(Object[] params ) 
             {  
                System.out.print("Hello world!");
                if (params[0] == null)
                   System.out.println();
                else
                   System.out.println(" And hello especially to "+params[0]);
             }  
           }
         }		
       );

Example 3. The default commands. 


       // Create a default parameter validator
       ParameterValidator pv = new ParameterValidator();
       // Create an empty command set
       Set<Command> cs = new HashSet();
       // Create the interpreter
       NaturalCLI nc = new NaturalCLI(cs, pv);
       // Add the commands that can be understood
       cs.add(new HelpCommand(cs));
       cs.add(new HTMLHelpCommand(cs));
       cs.add(new SleepCommand());  
       cs.add(new ExecuteFileCommand(nc));
       // Execute the command lines  
       nc.execute(args, 0);
