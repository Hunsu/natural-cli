Natural CLI

Natural CLI is a set of Java classes to provide command line interfaces with human readable sentences. It means, sentences like the following: 


   acme start daemon something
   acme send file /tmp/itsme.jpg to you@somehost.com
   acme start backup at 10:50

Main Features:

  • Java implementation.
  • Easy to code. Please, visit the examples page and the Javadocs.
  • Type checking for parameters and user defined types.
  • Optional parameters and tokens.
  • Build-id help command and others.
  • Can be compatible with Apache Commons CLI.

Limitations:

  • It's not a complier and only can be used very simple syntax. if you are interested in something more complex: tell me! ;-).
  • There are some combinations of parameters not allowed: there is no look ahead yet.
