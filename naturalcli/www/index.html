Natural CLI

Natural CLI is a set of Java classes to provide command line interface with human readable sentences. It means, sentences like the following: 


   acme start daemon <daemon_name>
   acme stop daemon <daemon_name>
   acme send file <file_name> to <email_address>
   acme start backup at 10:50

Main Features:

  • Java implementation.
  • Easy implementation. Please, visit the examples page.
  • Typed parameters with user defined types.
  • Optional parameters and tokens.
  • It is compatible with Apache Commons CLI.

Limitations:

  • It's not a complier and only can be used very simple syntax. if you are interested in something more complex: tell me! ;-).
  • There are some combinations of parameters not allowed: there is no look ahead.
