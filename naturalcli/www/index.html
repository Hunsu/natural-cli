Natural CLI

Natural CLI is an open source set of classes to be able to provide command line interface with human readable sentences. It means, sentences like the following::


    acme start daemon <daemon_name>
    acme stop daemon <daemon_name>
    acme send file <file_name> to <email_address>
    acme start backup at 10:50
